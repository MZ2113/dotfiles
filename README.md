# Neovim 配置 
 
## 运行环境 
* **Neovim 0.12.2** 
 
 
## 插件列表 
* **lazy.nvim**: 插件管理 
* **telescope.nvim**: 模糊查找 
* **nvim-lspconfig**: LSP 基础配置 
* **blink.cmp**: 代码补全 
* **catppuccin**: Catppuccin 主题方案 
* **toggleterm.nvim**: 内置终端集成 
* **nvim-autopairs**: 自动括号配对 
* **nvim-treesitter**: 语法高亮引擎 
 
 
## 自定义按键 (Keymaps) 
* `jk`: 退出插入模式 (`Esc`) 
* `<leader>oh`: 清除搜索高亮 
* `*`: 搜索当前单词但不跳转光标 
* `Alt + j/k`: 上下移动当前行或选中的代码块 
* `Ctrl + 方向键`: 调整分屏窗口大小 
* `<leader>ff`: 查找文件 (Telescope) 
* `<leader>fg`: 全局字符串搜索 (Telescope)
* `<leader>fb`: 切换已打开的缓冲区 (Telescope)
* `Ctrl + j`: 开关内置终端窗口
